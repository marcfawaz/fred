# Academy Use Cases

This folder contains demo scenarios organized by business theme.

## Recommended structure

- `README.md`: overview and setup
- `demos/<demo-name>/RUN_DEMO.md`: live demo prompts (copy/paste friendly)
- `demos/<demo-name>/*.yaml`: structured spec / regression script
- `knowledge/`: domain docs used during the demo
- `integration/`: links to runtime code and environment wiring
