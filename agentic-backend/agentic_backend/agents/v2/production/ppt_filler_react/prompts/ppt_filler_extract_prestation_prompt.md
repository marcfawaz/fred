Extrais les prestations financières depuis les extraits suivants.

Règles importantes :
- N'invente rien. Utilise uniquement les informations présentes dans les extraits.
- Les prix doivent être en euros, sous forme de nombres entiers.
- La charge est exprimée en unités d'œuvre.
- `prixTotal` d'une prestation = `prix × charge`.
- `prixTotal` global = somme de tous les `prixTotal`.
- Ne crée pas de catégories de prestations avec un coût nul.
- Si aucune information financière fiable n'est trouvée, laisse les noms vides et les montants à 0.
- Remplis uniquement les prestations pour lesquelles tu disposes à la fois d'un nom et de données financières concrètes.
